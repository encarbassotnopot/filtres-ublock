# filtres-ublock
Aquesta llista està enfocada a bloquejar els anuncis de pàgines catalanes/en català.

El projecte neix per comoditat, ja que cada vegada que obro un navegador nou perdo tots els filtres que he anat creant.
Dit això trobo que és d'utilitat pública i qualsevol contribució és benvinguda.

Ara per ara, no hi ha massa criteri de res: què bloquejar (elements molestos sí o no? O en una llista separada?), com ordenar els filtres...
Aniré prenent decisions a mesura que siguin necessàries.
